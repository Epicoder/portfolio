Greenfoot Virus Model Extension
Summer 2020, AP CSA

      For this projects my partners were Shahil Patel and Akshit Agarwal. This project was based around Michael Kölling's "Modeling a Virus Epidemic" Greenfoot tutorials which are found on Youtube. The task of this project was to add onto Kölling's pandemic scenario and create a real life scenario that could occur in the scenario. We decided to include masks for our scenario. What we did was create different actor that represented a masked person who wouldn't spread disease. This actor would also be unable to receive any transmission because they were covering their face. Our primary reason for including a "masked" actor was to illustrate how masks are essential in controlling the pandemic. We wanted to spread a common message that being the importance of wearing masks.

NYSE Data Analysis Using Python (Unit 2 Project)
January 2021 - February 2021, AP CSA

      My partners for this project were Keenan Kalra and Shahil Patel. For this project, we were instructed to create a fun & interesting program that used an unique dataset. In addition, we were allowed to work with any programming language of our choice. Our group eventually ended up working with Python since we were all familiar with the language due to the skills we had learned in Computer Science Principles. For our dataset, we decided to choose the Yahoo Finance Library and create three separate programs with it. Our first program was a stock price searcher, where the user could input any ticker (Stock Symbol) of their choice and the daily listings would pop up. Our second program was where the user could compare the cumulative returns of three different stocks and the program would print out a graph showing the differences over time. Finally, our third program was one where the user would input their desired stock and price and the system would send an email to them once their criteria was met.
